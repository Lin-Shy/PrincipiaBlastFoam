# 基于LLM生成搜索策略的知识图谱检索 - 单页总结

## 🎯 核心创新

**利用LLM动态生成针对知识图谱的精确搜索策略，实现智能化、自适应的技术文档检索**

---

## 📐 方法流程（4步）

```
用户查询 → LLM生成搜索策略 → 本地图谱搜索 → 返回Top-K结果
```

### 1️⃣ LLM策略生成
- 输入: 用户查询 + 知识图谱结构摘要
- 输出: JSON格式搜索策略
  - 节点类型过滤 (Case/File/Variable)
  - 属性匹配条件 (name/path/type)
  - 关键词列表
  - 排序偏好

### 2️⃣ 本地搜索执行
- 标签匹配: +5.0分
- 属性匹配: +3.0分  
- 关键词匹配: +1.0-2.0分

### 3️⃣ 智能排序
- 基于多维度评分
- 结合策略偏好
- 筛选Top-K节点

### 4️⃣ 内容检索
- 节点属性
- 关联关系
- 完整文件内容

## 🌟 实际案例

**查询**: "What turbulence model is used in the axisymmetricCharge case?"

**LLM生成策略**:
```json
{
  "search_criteria": {
    "node_labels": ["Case", "File", "Variable"],
    "property_filters": {
      "name": "*turbulence*",
      "path": "*axisymmetricCharge*"
    },
    "keyword_search": ["turbulence", "model"]
  },
  "ranking_criteria": {
    "prefer_node_types": ["File", "Variable"]
  }
}
```

**检索结果**:
1. ✅ turbulenceProperties (File) - 目标文件
2. ✅ turbulenceModel (Variable)
3. RASModel (Variable)
