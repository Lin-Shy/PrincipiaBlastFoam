# 求解器: blastEulerFoam (含颗粒物的欧拉-欧拉多相流求解器)

## 1. 核心功能

`blastEulerFoam` 用于模拟爆炸场景中的气固两相流问题。它采用欧拉-欧拉方法，将气体（连续相）和颗粒物（离散相）都视作可以相互贯穿的连续介质。这种方法适合于模拟颗粒物体积分数较高的场景。

## 2. 适用场景

- **含尘环境中的爆炸**: 模拟在充满沙尘、粉尘的空气中发生爆炸，分析颗粒物对冲击波传播的影响。
- **爆炸驱动颗粒物**: 模拟爆炸将地面沙土或预设的颗粒物扬起并形成颗粒云的过程。
- **反应性颗粒**: 模拟金属粉末等可燃颗粒物在冲击波作用下的燃烧或爆炸。

## 3. `blastFoam_tutorials` 中的典型案例

- `reactingParticles`: 模拟了反应性铝颗粒在爆炸场中的行为。
- `shockTube_polydisperse`: 在激波管中模拟了多种不同尺寸的颗粒（多分散颗粒）与激波的相互作用。

## 4. 关键目录结构与文件

`blastEulerFoam` 的案例结构在 `blastFoam` 的基础上，为颗粒相增加了专属的物性、初始条件和边界条件文件。

```
.
├── 0/
│   ├── p, T, U                     # 气体相的初始条件
│   ├── alpha.gas                   # 气体相的体积分数
│   │
│   ├── U.particles                 # **新增**: 颗粒相的速度场
│   ├── T.particles                 # **新增**: 颗粒相的温度场
│   └── alpha.particles             # **新增**: 颗粒相的体积分数
│
├── constant/
│   ├── cloudProperties             # **核心文件**: 定义颗粒相的各种属性，如名称、密度、直径、热力学模型、相间作用力模型（如曳力）等。
│   ├── thermophysicalProperties    # 气体相的热物理属性
│   ├── turbulenceProperties        # 气体相的湍流模型
│   └── polyMesh/
│
└── system/
    ├── controlDict                 # 设置求解器为 `application blastEulerFoam;`
    ├── fvSchemes                   # 包含气体相和颗粒相的离散格式
    └── fvSolution                  # 包含气体相和颗粒相的求解算法
```

### 总结:
使用 `blastEulerFoam` 的关键在于理解和配置 `constant/cloudProperties` 文件，它决定了颗粒物的所有物理和化学特性。同时，需要在 `0` 目录下为颗粒相提供独立的初始条件。这使得案例的设置比标准的 `blastFoam` 更为复杂。
